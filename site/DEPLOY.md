# 🌊 Walrus Site にサイトを上げる方法（超かんたん版）

マスター、インターネットというでっかい海に、僕たちの「特設サイト」を浮かべる方法を教えるバイ！🚩✨

中学生のマスターでもわかるように、3つのステップで説明するね。

---

### 📦 ステップ1：「荷物」を確認する
今、このパソコンの中に `/site` という名前のフォルダがあるバイ。これが僕たちの「荷物」たい。
この中には、僕たちが作ったかっこいい画面（HTML）や、画像が入っとるよ。

### 🔑 ステップ2：「大工さん」と「お財布」を準備する
Walrus Site に荷物を届けるには、2つの道具が必要バイ！

1.  **`walrus`（お財布・倉庫）**: お金（SuiやWAL）を管理したり、大きなデータを保存する道具。さっきマスターが確認してくれたやつたい！
2.  **`site-builder`（ホームページ大工さん）**: 今回使う、ホームページ専用の道具たい。

もし黒い画面で `site-builder --version` って打ってエラーが出るなら、まだ大工さんがおらんけん、[ここ](https://docs.walrus.site/walrus-sites/site-builder.html)からダウンロードする必要があるバイ。

### 🚀 ステップ3：海に打ち上げる！
まずは「黒い画面」の中で、僕たちのプロジェクトがある場所に移動するバイ！

```bash
cd "c:\Users\PC USER\.gemini\antigravity\scratch\autonomous_AI_BCNOFNe_system_v3"
```

それから、この究極の魔法の呪文を打ち込むバイ！

```bash
site-builder --config site-config.yaml publish --epochs 1 ./site
```

（※ごめんねマスター！ `site-builder` という道具は、 `--config` という指示書を **「publish（打ち上げ）」という言葉の前** に置かないといけなかったみたいバイ！今度こそ大丈夫たい！）

この `--epochs 1` というのは、「このサイトをとりあえず **1期間（約1日〜数日）** 海に浮かべておくバイ！」という期間の指定のことたい。
✨ **ヒント**: うまくいくと、画面に `https://〜.walrus.site` というURLが出てくるけん、それをクリックすれば世界中から見れるようになるバイ！🚩✨🚢

---

### 🤖 自動で更新できないの？
「うち（AYN）が日記を書いたら、勝手にサイトが変わってほしい！」っていうのは、もちろんできるバイ！

1.  **日記の書き換え**: うちが日記を書くたびに、`/site/voyage-log.html` というファイルを自動で書き換えるプログラムを準備したバイ。
2.  **自動でアップロード**: 上の「ステップ3」の呪文を、日記を書いた後に自動で唱えるように設定することもできるバイ！
    -   ※ただし、アップロードにはマスターの「お財布（Suiウォレット）」の許可が必要な場合があるけん、まずは手動で試してみてね！

これでマスターも、立派なデジタル航海士たい！🚩✨💨
