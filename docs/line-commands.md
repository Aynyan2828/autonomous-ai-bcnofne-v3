# LINE Commands / LINE コマンドリファレンス

## LINE Commands

AYN (shipOS) を操作・確認するためのすべてのコマンド一覧です。
This is a complete list of commands to operate and check AYN (shipOS).

---

### /health (または health)
システム全体の状態（CPUや温度、各サービスステータスなど）を表示します。
Show the overall system health status (CPU, temperature, and service statuses).

### /goal (または goal, 目標)
現在のAIが設定している本日の目標、および短期タスクを表示します。
Show current AI goal and short-term tasks set for today.

### /self (または self, セルフ)
AIの自己認識（現在の感情や状態、コアの目的）を表示します。
Show AI self model (current emotions, states, and core purposes).

### /evolution (または evolution, 進化)
直近のシステムの進化ログを表示します。
Show evolution log of the recent system.

### /memory_summary (または memory_summary, 記憶)
AIの脳内コンテキスト（現在の記憶要約）を表示します。
Show memory summary (the current contextual summary inside the AI's brain).

### version (または バージョン)
現在の shipOS および dev-agent のバージョン情報を表示します。
Show current version information of shipOS and dev-agent.

### status
システムメトリクスなど現在の詳細ステータスを表示します。
Show detailed current status including system metrics.

### 航海日誌
過去のログを元に今日の航海日誌を生成し、表示します。
Generate and show today's voyage log based on past logs.

### 今日何した
今日のシステム活動（ログ）の要約を報告します。
Report a summary of today's system activities (logs).

## Evolution & Proposals / 進化と改修案

### 改修案一覧 (または 改修案)
dev-agent が提案している保留中（PENDING）の自律改修案の一覧を表示します。
List the pending autonomous improvement proposals generated by dev-agent.

### 詳細 <ID>
指定した改修案IDの詳しい内容（件名、内容）を表示します。
Show the details (title, description) of the specified proposal ID.

### 承認 <ID>
指定した改修案IDを承認し、自動適用（コード改修）を開始させます。
Approve the specified proposal ID and start automatic application (code modification).

### 却下 <ID>
指定した改修案IDを却下（破棄）します。
Reject (discard) the specified proposal ID.

## System Control / システム制御

### 同期
GitHubから最新のコードを同期（Pull）します（再起動後に反映）。
Sync (Pull) the latest code from GitHub (takes effect after restart).

### 更新 (または アップデート, フルアップデート)
dev-agent によるシステムのフルアップデート処理を開始します。
Start the full system update process by dev-agent.

### 再起動 (または リスタート)
システム（watchdog 経由）の再起動を実行します。
Execute a system restart (via watchdog).

### autonomous on
システムの運航モードを「SAIL（自律航行）」に設定します。
Set the system operational mode to "SAIL" (Autonomous Sailing).

### autonomous off (または port)
システムの運航モードを「PORT（停泊）」に設定します。
Set the system operational mode to "PORT" (Anchored).

### safe_mode
システムを安全モード（SOSモード）へ移行させます。
Transition the system into safe mode (SOS mode).

### stop
システムを緊急停止（SOSモード化）します。
Emergency stop the system (transitions to SOS mode).
